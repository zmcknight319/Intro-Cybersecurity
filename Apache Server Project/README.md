# Apache Web Server Project

## Project Overview

This project demonstrates the installation, configuration, and verification of an Apache web server on Ubuntu Linux. The lab was completed as part of my Intro to Cybersecurity coursework.

---

## Objectives

- Install Apache HTTP Server
- Verify Apache is running
- Enable the Apache service
- Access the default web page
- Practice Linux command-line administration

---

## Environment

| Component | Details |
|-----------|---------|
| Operating System | Ubuntu Linux |
| Server | Apache2 |
| Terminal | Bash |
| Platform | Virtual Machine |

---

## Skills Demonstrated

- Linux command execution
- Package installation
- Service management
- Apache web server deployment
- Basic troubleshooting
- Command-line administration

---

## Commands Used

```bash
sudo apt update
sudo apt install apache2
sudo systemctl start apache2
sudo systemctl enable apache2
sudo systemctl status apache2
hostname -I
```

---

## Verification

Apache was successfully installed and verified by accessing the default Apache webpage from a web browser.
